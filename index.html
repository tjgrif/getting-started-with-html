What Makes up the Web
The Web is simply a number of servers (computers) connected to each other. They "speak" to each other through HTTP (referred to as a 'protocol').
HTML is the language that allows a browser on a computer to interpret and order the files from other serviers on the internet.

HTML
HTML stands for Hypertext Markup Language.
Elements within HTML are called "tags".
"Tags" instruct the computer to show everthing: from whether a word is bold, italicised, or a hyperlink.

Inline v Block
At a simple level, laying-out webpages involves either letting objects or words simply exist on the page "inline" or have an invisible box around them "block".
